Info:

Repository-config uses for define script parametrs like: dependencies, description, script-configurations. Note: Write the configuration neatly, but if you do something wrong, everything will break down!


below there are examples:

REP_Desc = "Description"

Set script description

REP_Dependency = {"Dep1", "Dep2" ...}

Set script dependencies. You can also use packages with configs. below the example:

REP_Dependency = {"ScriptConfig1.ini", "Luasocket"}
